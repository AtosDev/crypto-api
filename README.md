# cryptonews-scrape-api


## To run this project

You can run this project on your local machine. Just pull it down and do the following:

### `npm i`

1. Run the command above to install all the packages

### Start the Project

2. Now run the server:

```bash
npm run dev
```

3. Open [http://localhost:3000/cryptonews](http://localhost:3000/cryptonews) with your browser to see the results of the scrape in your terminal. You can also use certain paramters to get news from individual sources.

** _You must have [Json Viewer](https://chrome.google.com/webstore/detail/json-viewer/gbmdgpbipfallnflgajpaliibnhdgobh) installed in chrome extensions, to view the response in a good manner _**
